Startup Scanner helps students discover **startups that are worth applying to** using live data and AI analysis.

---

Startup Scanner combines **real-time data + AI** to surface high-potential startups tailored to you.

You enter:

* Industry
* Location
* Role type
* Interests

The app:

* Aggregates startup signals from **live RSS feeds**
* Uses AI to evaluate each company
* Ranks startups based on:

  * Growth
  * Relevance
  * Fit

## 🧪 How It Works

1. Enter search criteria *(e.g. AI internships in London)*
2. Run a **live search**
3. View ranked startup results
4. Click a company for deeper insights

---

## 📦 Local Setup

```bash
# Clone the repo
git clone https://github.com/your-username/startup-scanner.git

# Navigate into project
cd startup-scanner

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```
